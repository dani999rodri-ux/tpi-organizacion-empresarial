# Descripción del Proyecto
Este proyecto consiste en un simulador de chatbot por consola desarrollado en Python. Automatiza el proceso administrativo de solicitud de vacaciones de una organización, vinculándose de manera dinámica con una base de datos persistente simulada en Excel. La lógica del sistema responde estrictamente al diseño de compuertas y estados modelados previamente en la notación BPMN 2.0.
